# 📊 Morning Brief

Daily intelligence briefing covering markets, geopolitics, portfolio analysis, regulatory deadlines, and SAP intelligence. Generated by Claude with live data.

## Coverage
- **Geopolitics** — Iran war, US tariffs, LATAM developments
- **Markets** — S&P 500, Nasdaq, Brent, Gold, Merval, Ibovespa, FX
- **Portfolio** — Concentration, factor exposures, stress scenarios, recommendations
- **Regulation** — EU AI Act (HRAI), NIS2 Germany, DORA, CRA
- **SAP Intelligence** — Enterprise risk, CCB pipeline, Joule AI Act exposure

## Structure
```
briefs/YYYY-MM-DD.html   ← individual daily briefs
index.html               ← always the latest brief
archive.html             ← full archive with links
```
